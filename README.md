# 🧠 Placify

Live Demo: **https://placify-jade.vercel.app/**

Placify is a **full-stack Resume ATS Analyzer & Job Opportunity Platform** built with React (Vite) frontend and Node.js + Express backend.  
It helps students and job seekers analyze their resumes against specific job roles using keyword matching, smart section detection, and actionable suggestions — just like a real ATS (Applicant Tracking System).

---

## 📦 Features

### ⭐ ATS Resume Analyzer
✔ Upload resume (PDF or DOCX)  
✔ Target specific roles (Frontend, Backend, Full Stack, DevOps, Data Analyst, Java)  
✔ Get ATS compatibility scores  
✔ See matched / missing skills  
✔ Receive improvement suggestions

---

### 💼 Job Opportunities (UI Placeholder)
✔ Opportunity cards  
✔ Skill tags  
✔ Connect buttons

---

### 🗂 Student Dashboard (Future)
✔ Stats & profile overview  
✔ Quick actions
